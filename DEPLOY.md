# TVInternetSBB Bot - Deployment Guide

## Cloud Hosting Options:

### 1. PythonAnywhere (Recommended)
- Free tier available
- Always-on tasks
- Easy setup

### 2. Heroku
- Free with credit card
- Auto-deploy from GitHub
- Easy scaling

### 3. Replit
- Always-on option
- Simple interface
- Free tier

## Environment Variables:
- BOT_TOKEN: Your Telegram bot token
- ADMIN_CHAT_ID: Your Telegram chat ID for notifications
