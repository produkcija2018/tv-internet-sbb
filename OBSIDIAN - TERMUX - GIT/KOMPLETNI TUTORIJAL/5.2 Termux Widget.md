### Instalirajte Termux:Widget:

1. **Google Play/F-Droid** → **"Termux:Widget"**
2. **Instalirajte** pored Termux aplikacije
### Kreiranje Widget Skripte:
# Kreirajte folder za widget skripte
mkdir -p ~/.shortcuts

# Kreirajte widget skriptu
echo '#!/bin/bash
cd ~/storage/shared/Obsidian-Vault
./sync.sh
termux-vibrate -d 300 2>/dev/null' > ~/.shortcuts/sync-notes

chmod +x ~/.shortcuts/sync-notes
